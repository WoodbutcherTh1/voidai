What is VoidAI?
VoidAI is a next-generation, multi-modal AI platform built for developers, creators, and
power users. It brings together the most powerful AI capabilities into one seamless,
beautifully designed web app — accessible from any browser, on any device.
Bring your own API key. No subscriptions. No limits imposed on you. Just pure AI power.

Getting Started
# Clone the repository
git clone https://github.com/yourusername/voidai.git
# Navigate to project
cd voidai
# Install dependencies
npm install
# Set up environment variables
cp .env.example .env.local
# Run development server
npm run dev
Open http://localhost:3000 — enter your API key — and step into the Void.
<br/>
Environment Variables
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
All AI API keys are entered by the user in-app. No keys needed in .env.
